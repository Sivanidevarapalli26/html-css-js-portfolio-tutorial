Projects

Deep Learning Classifier for LLM Identification

BERT-based classifier to attribute responses to the source LLM.

Built and fine-tuned a BERT classifier to distinguish outputs from GPT-2, GPT-Neo, Falcon, and OPT, achieving ~81% accuracy on a curated set of 35,028 prompt–response pairs sourced from 8.7k prompts. The training pipeline used AdamW, dropout, early stopping, and an error-analysis dashboard to surface failure modes and guide dataset refinements.

Tech: PyTorch, Hugging Face, scikit-learn, Python

Distributed Parallel File System

Token-based concurrency with sequential consistency.

Architected a distributed file system spanning multiple data servers with a centralized metadata service to enforce sequential consistency via token management. Implemented an LRU write-back cache to reduce I/O, and optimized RPC paths with gRPC and efficient serialization to keep typical operation latency under 10 ms.

Tech: C++/gRPC, protobuf, Linux, multithreading

Flight Delay Analysis (1998–2002)

Merged U.S. BTS flight data with NOAA weather.

Unified historical BTS flight logs with NOAA weather data, engineered features, and pruned outliers to build reliable models of delay risk. Trained XGBoost and SVM with grid search and 5-fold CV, reaching ~90% accuracy, and identified a ~12% rise in average delay rates in the post-9/11 period.

Tech: Python, Pandas/NumPy, scikit-learn, Matplotlib

Evaluating Logical Reasoning in LLMs

Stress-tests for reasoning brittleness under misleading prompts.

Authored the Faulty_Q dataset to probe logical consistency and sensitivity to adversarial prompt phrasing, observing baseline accuracy of ~92% with drops of ~17% under misleading prompts across models including GPT-4 and Claude. Built a reusable evaluation harness to reproduce results and compare model families.

Tech: Python, evaluation pipelines, experiment tracking
